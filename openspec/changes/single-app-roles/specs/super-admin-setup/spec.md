## 新增需求

### 需求：预设超级管理员账号
系统必须预设一个超级管理员账号，默认密码为 123456。

#### 场景：初始化超级管理员
- **当** 系统首次启动
- **则** 自动创建超级管理员账号（用户名：admin，密码：123456）

#### 场景：首次登录修改密码
- **当** 超级管理员使用默认密码首次登录
- **则** 系统要求必须修改密码

### 需求：超级管理员权限
超级管理员必须拥有系统的最高权限。

#### 场景：超级管理员访问所有功能
- **当** 超级管理员登录
- **则** 系统允许访问所有管理功能

#### 场景：创建普通管理员
- **当** 超级管理员创建新的管理员账号
- **则** 系统创建新的 admin 角色用户
