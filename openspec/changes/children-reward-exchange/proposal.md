## 需求背景

孩子们需要一个游戏化的奖励系统来记录日常学习和生活中的成就，通过积分兑换机制鼓励积极行为。所有奖励和兑换都需要管理员审核，确保透明度和适当性。

## 变更内容

- 构建微信小程序让孩子登录、记录奖励、浏览奖品、提交兑换申请
- 构建管理员小程序审核和批准/拒绝奖励和兑换申请
- 使用 CloudBase MySQL 数据库持久化存储
- 实现微信分享功能，方便管理员快速审核待处理申请

## 功能模块

### 新增功能

- `user-auth`: 用户认证系统，支持孩子账号和管理员账号，基于角色的访问控制
- `reward-recording`: 孩子提交奖励申请及描述，管理员审核批准/拒绝
- `prize-catalog`: 展示可用奖品及所需积分和库存
- `prize-redemption`: 孩子提交兑换申请，管理员审核批准/拒绝
- `admin-review`: 管理员仪表盘，查看、筛选、处理待审核申请
- `share-to-admin`: 生成待处理申请的分享链接，实现一键管理员审批

## 影响范围

- 新增 CloudBase MySQL 数据库表：用户、奖励、奖品、申请表
- 微信小程序（孩子版）：奖励记录和奖品兑换功能
- 微信小程序（管理员版）：申请审核功能
- CloudBase 云函数：后端 API 逻辑
