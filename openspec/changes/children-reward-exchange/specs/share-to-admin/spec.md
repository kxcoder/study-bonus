## 新增需求

### 需求：生成分享链接
孩子用户必须能够为待处理的申请生成微信可分享的链接。

#### 场景：生成分享链接
- **当** 孩子在待处理申请上点击"分享审核"
- **则** 系统生成包含申请 ID 和类型的可分享链接

### 需求：管理员接收分享链接
管理员用户必须能够打开分享链接并跳转到预加载申请的审核页面。

#### 场景：管理员打开分享链接
- **当** 管理员从微信打开分享链接
- **则** 系统跳转到预加载申请的审核页面

### 需求：非管理员打开分享链接
非管理员用户打开分享链接必须被重定向到首页并显示消息。

#### 场景：孩子打开管理员链接
- **当** 孩子用户打开分享的管理员链接
- **则** 系统显示"此链接仅限管理员"并重定向到首页

### 需求：分享页一键批准
管理员在分享页面查看申请必须能够一键批准或拒绝。

#### 场景：分享页快速批准
- **当** 管理员在分享页点击"批准"
- **则** 系统处理申请并显示成功消息

#### 场景：分享页快速拒绝
- **当** 管理员在分享页点击"拒绝"
- **则** 系统提示输入拒绝原因，然后处理拒绝
